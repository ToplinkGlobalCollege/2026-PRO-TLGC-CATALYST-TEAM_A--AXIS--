## Robot Improvements Made in May
We started by building the basic robot body and wheels. The first version used one large motor for driving, one medium motor for steering, two ultrasonic sensors to avoid obstacles, and a gyro sensor to count laps. Components were placed in standard positions to test basic functionality. Later, we changed the design to use two medium motors for driving and one medium motor for steering. We moved the EV3 closer to the steering motor to make the robot smaller and more balanced. The EV3 stayed turned sideways, which kept the wires clear and easy to manage. Finally, we changed the sensors to use one ultrasonic sensor to avoid obstacles, a gyro sensor to count laps and fix the steering, and a color sensor to also count laps.
